### free

说明: 查看内存和swap

语法: `free [选项]`

* `-h` human readable

### top

输出说明

| PID    | USER   | PR     | VIRT     | RES      | SHR      | S    | TIME+ | COMMAND |
| ------ | ------ | ------ | -------- | -------- | -------- | ---- | ----- | ------- |
| 进程ID | 用户名 | 优先级 | 虚拟内存 | 常驻内存 | 共享内存 | 状态 |       |         |

> VIRT: 进程申请的内存
>
> RES: 进程及其调用进程使用的内存
>
> SHR: 调用其他进程的内存

快捷键

`M` 内存

`P` CPU

`1` 展开或缩起CPU
