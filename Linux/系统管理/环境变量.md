### env

说明: 查看Linux中所有环境变量

### export

说明: 将变量提升为全局变量

<center>重要的环境变量</center>

| 变量名称 | 作用                         |
| -------- | ---------------------------- |
| HOME     | 用户家目录                   |
| SHELL    | 用户使用的shell              |
| HISTSIZE | 历史命令条数                 |
| MALL     | 邮件保存路径                 |
| LANG     | 系统语言                     |
| RADOM    | 生成随机数                   |
| PATH     | 解释器搜索用户执行命令的路径 |
