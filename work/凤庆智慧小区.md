# 凤庆智慧小区

## API

### 用户

#### 获取token

```js
request = {
    code
}

response = {
    token
}
```

#### 获取小区列表

```js
response = [
    {
        hood_id,
        hood_name: '小区名'
    }
]
```

#### 上传用户信息

```js
reqest = {
    token,
    img: '用户照片',
    hood_id: '小区id',
    name: '姓名',
    id_card: '身份证',
    phone: '电话',
    verify_code: '验证码'
}
```

#### 上传用户微信信息

```js
request = {
    avatar: '头像',
    nickname: '昵称'
}
```

#### 获取用户信息

